# US - Useful Summary

### 개발 배경
현대 사회에서는 교재나 노트 필기와 같은 텍스트 자료가 많이 사용되고 있다.  
그러나 이러한 자료들은 보통 방대하고 상세한 내용을 포함하고 있어서 효과적으로 활용하기 어렵다.  
따라서 이러한 문제를 해결하기 위해 OCR 기술을 이용하여 교재나 노트 필기의 텍스트를 인식하고, 이를 요약하는 어플을 개발하였다.
